## 📌 O‘rnatish

1️⃣ Reponi klonlash:
   ```sh
   git clone https://github.com/username/Safo-non.git
   ```
2️⃣ Loyihani sozlash:
   ```sh
   cd Safo-non
   npm install  # yoki pip install -r requirements.txt
   ```
3️⃣ Serverni ishga tushirish:
   ```sh
   npm run serve
   ```

## 🛠 Ishlatilgan Texnologiyalar
